# SmileCare Dental Clinic — Premium Next.js Website

A modern, premium dental clinic website with outstanding UI/UX design.

## Tech Stack
- **Next.js 14** — React framework
- **Tailwind CSS** — Utility-first styling  
- **Framer Motion** — Smooth animations

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run development server
```bash
npm run dev
```
Open http://localhost:3000

### 3. Build for production
```bash
npm run build
npm start
```

## Pages
- `/` — Home (hero with cinematic animation, services, testimonials, FAQs, blog)
- `/about` — About Us (doctor profile, career timeline, team)
- `/services` — Services (6 service cards with details and booking)
- `/appointment` — Booking Form (3-step form with validation)
- `/contact` — Contact (form, map, click-to-call, WhatsApp)

## Customization
- Edit clinic info in each page component
- Update colors in `tailwind.config.js`
- Replace placeholder doctor info in `/pages/about.js`
- Update phone/email in `/components/Navbar.js`, `/components/Footer.js`
- Update Google Maps embed URL in `/pages/contact.js`
